## how to install

clone this repo

    git clone https://github.com/ahdinosaur/dinOS.git

install live-build

    sudo aptitude install live-build

build .iso

    ./build.py
    
copy the .iso to your lucky flash drive

    dd if=binary.hybrid.iso of=/dev/sd#
